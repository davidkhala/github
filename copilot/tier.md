# premium requests
> For instance, one Copilot coding agent session or a code review assignment typically counts as one premium request unless using a model with a multiplier.
- Using Copilot Chat with models with **Model Multipliers**
  - models without multipliers: [GPT-4o, GPT-4.1]
- Conducting code review (assigning Copilot as a reviewer or reviewing in your IDE).
  - Decided by GitHub
- Multi-file edits or “agentic” coding (Copilot Coding Agent).
  - Visual Studio Code has a feature called **agents** that allow you to ask questions using a specific context
  - To use an **agent**, prefix your question with the agent
  - e.g. `@workspace how can I package this project?` is aware of your entire workspace
  - `@terminal` agent helps you chat with GitHub Copilot to interact with the terminal.
  - `@file`, `@directory`
- Making requests via CLI or certain extensions.
  - Decided by GitHub
- Creating apps with Spark.
  - 4 premium requests per prompt (fixed).
- Using Copilot Spaces.
